### 3_OX퀴즈.py

- 처음에 O +=1 아니면 그대로로 문제를 이해하고 작성했는데 값이 적게 나왔음
- 연속된 OO를 고려하지 않은 실수
- 연속된 'O'가 나오면 +2,+3,+4 가나오는걸 발견하고
- 새로운 변수C를 C=1로 잡아주고
- O를 더하면 C+=1하고
- 아니면 c=1로 조건을 변경하고 실행했더니 됐음
